## PPT整体设计方案（20页，8分钟）

### 推荐画幅与节奏

- 画幅：16:9（1920×1080）
- 8分钟建议：前6页讲"为什么必须做"；中间8页讲"我们怎么做、做得多好"；最后6页讲"怎么落地、怎么赚钱、谁来做、未来去哪"

### 统一视觉风格（建议你直接照抄成PPT母版）

- 关键词：新中式 + 科技感 + 纸纹质感（对应你产品里 ink/paper 的气质）
- 主色：墨黑 #111111 / 纸白 #F6F2EA / 朱砂红 #C43C2E / 科技蓝 #2B59FF
- 字体（PPT可用）：标题用"思源宋体/方正书宋/宋体加粗"，正文用"思源黑体/苹方/微软雅黑"；英文用"Times New Roman + Source Sans（或系统默认）"
- 版式：每页不超过 1 句主标题 + 3 个要点；尽量"左文右图"或"上图下文"；关键数字用大号字体
- 图片统一要求（给文生图用）：不要出现文字/水印/logo；留白可叠字；同一套色调（墨黑+朱砂+蓝）

### 全局通用负面词（每页都能复用）

负面提示词：不要文字，不要水印，不要logo，不要二维码，不要畸形手指，不要多余肢体，不要模糊，不要低清，不要过曝，不要血腥暴力，不要政治敏感标识。

cd server
npx tsx scripts/test-api.ts

```bash
cd server
npx tsx scripts/generate-ppt-images.ts
```

3. **查看生成结果**
- 图片位置: `ppt/images/`
- 生成记录: `ppt/images/generation-record.json`

### 技术细节

- **API Key**: 使用专用 API `7041e3a8-18c7-4949-8641-7bbd82cc17ab`（不影响项目其他功能）
- **水印设置**: `watermark: false`（所有图片无水印）
