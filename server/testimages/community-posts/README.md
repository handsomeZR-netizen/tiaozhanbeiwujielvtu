# 社区灵感帖子图片资源

## 目录说明

本目录包含社区灵感功能的所有内容资源和生成工具。

### 文件列表

- `CONTENT_PLAN.md` - 完整的内容策略文档，包含 12 条帖子的详细设计
- `community-posts-data.json` - 帖子数据的 JSON 格式，可直接用于前端集成
- `generation-record.json` - 图片生成记录（生成后自动创建）
- `post-*.jpg` - 生成的帖子配图（生成后自动创建）

## 使用流程

### 1. 审核内容文案

查看 `CONTENT_PLAN.md`，确认以下内容：
- 12 条帖子的标题、正文、标签是否符合要求
- 文化提示是否准确
- 文生图提示词是否合适

### 2. 生成图片

确认文案无误后，运行生成脚本：

```bash
cd server
npm run generate:community-images
```

或直接运行：

```bash
cd server
npx tsx scripts/generate-community-images.ts
```

### 3. 检查生成结果

- 查看 `generation-record.json` 了解生成状态
- 检查生成的图片质量和内容是否符合预期
- 如有失败，可单独重新生成

### 4. 集成到前端

使用 `community-posts-data.json` 中的数据更新 `CommunityPage.tsx`：

```typescript
import postsData from '../../../server/testimages/community-posts/community-posts-data.json';

// 在组件中使用
const posts = postsData.posts;
```

## 内容策略概览

### 6 大内容支柱

1. **城市文化微叙事** - 真实游客讲述城市细节
2. **非遗/传统工艺体验** - 手作体验与文化传承
3. **城市夜间经济/夜景** - 夜市、灯会、夜游路线
4. **饮食文化与礼仪** - 菜系、点单体验、文化差异
5. **城市公共空间与生活方式** - 公园、广场、晨练
6. **入境便利体验** - 支付、交通、友好度评价

### 真实感写作规范

每条帖子包含：
- ✅ 时间戳（如 2026/01/22 08:40）
- ✅ 真实花费
- ✅ 小插曲（迷路、沟通、意外好评）
- ✅ 一句个人感受（强调主观）
- ✅ 文化提示（避免空洞宣传）

### 文生图技术规范

- **API**: doubao-seedream-4-5-251128
- **尺寸**: 2K（约 2048x2048）
- **语言**: 中文提示词
- **质量关键词**: 4K超高清画质、纪实摄影风格、电影构图
- **风格**: 真实感优先，温暖色调，文化细节
- **避免**: 过度滤镜、不真实摆拍、商业化视觉

## 注意事项

⚠️ **生成前必读**：
1. 确保已配置 `ARK_API_KEY` 环境变量
2. 使用 doubao-seedream API，支持中文提示词
3. 豆包 API 有调用频率限制，脚本已设置 3 秒间隔
4. 图片生成需要时间，预计总耗时 10-15 分钟（16 张图片）
5. 生成失败的图片可以单独重新生成
6. 图片 URL 有时效性，建议及时下载保存

⚠️ **内容合规**：
- 所有内容符合文旅融合政策导向
- 强调真实体验，避免空洞宣传
- 尊重文化多样性，避免刻板印象
- 适合国际传播，友好外宣语境

## 技术支持

如遇问题，请检查：
1. 环境变量配置是否正确
2. 网络连接是否正常
3. API 额度是否充足
4. 查看 `generation-record.json` 中的错误信息
